Wzorce UI do kopiowania 1:1

To jest źródło prawdy dla nowych elementów. Modal bierz z ROZRYS, pola wpisywania z Inwestor, a launchery / stopki z zaakceptowanych wzorców ROZRYS. Nie tworzyć lokalnych wariantów, jeśli odpowiednik już istnieje.

Modal / shell / header / X

Kopiować 1:1 przez klasy: panel-box panel-box--rozrys, panel-box__head, panel-box__title, panel-box__close.

Wzorcowy modal

Pole wpisywania

Pola tekstowe / liczbowe w formularzach mają kopiować shell i rytm z Inwestor. Nie robić „podobnych” inputów lokalnie.

Launcher / picker

Widoczny wybór ma wyglądać jak launcher z aplikacji. Native select zostaje źródłem prawdy i siedzi pod spodem.

Stopka akcji

Dół modala kopiować z ROZRYS: rozrys-picker-footer, prawa grupa akcji, mniejsze przyciski i powietrze pod spodem.

Przyciski nie mają tworzyć ciężkiego pasa przez całą szerokość. Mają siedzieć po prawej stronie w lekkiej stopce.
